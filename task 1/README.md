# 大二组 Github使用与opencv基础

## 1、环境要求

（1）系统环境：ubuntu 18.04+opencv 3

（2）编译环境：不使用IDE，直接使用cmake编译源码并运行。

## 2、任务要求

（1）学习使用github，将任务仓库克隆到本地，进行相应的作业更改之后，对源仓库pull request。

（2）掌握opencv基础，编写main.cpp，实现打开摄像头并对采集图像进行颜色反转(255-对应通道值)的功能。

## 3、提交要求

提交两份文件：

（1）CMakeList.txt

（2）main.cpp

## 4、评分标准

（1）使用cmake单独编译

（2）提交时间

（3）颜色反转正确性
